# Finance Flow - API Gateway 🚀

This is the robust backend for the Finance Flow ecosystem. Built with a focus on scalability and clean data management, this API handles all financial transactions and category logic.

### 🛠️ Tech Stack
- **Server:** Node.js & Express.js
- **Database:** MongoDB (via Mongoose ODM)
- **Security:** Dotenv for environment variables & CORS for cross-origin requests.
- **Tools:** Postman for API testing.

### 📋 Features
- **RESTful API:** Clean endpoints for GET and POST transactions.
- **Data Validation:** Strict schema definitions for amounts, descriptions, and categories.
- **Persistence:** Connected to MongoDB Atlas for cloud-based data storage.

### ⚙️ How to run
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file with your `MONGO_URI` and `PORT`.
4. Run `npm start` or `node server.js`.
